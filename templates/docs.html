Getting Started

Welcome to the Telegram Group Manager Bot documentation. This guide will help you understand how to use the bot effectively in your Telegram groups.

Adding the Bot to Your Group

  1. Start a conversation with the bot by clicking here or searching for @kadhannaaCenter_bot in Telegram
  2. Send the /start command to initialize the bot
  3. Open your Telegram group
  4. Click on the group name at the top to access group info
  5. Tap "Add member"
  6. Search for and select the bot
  7. Once added, promote the bot to administrator for full functionality

Initial Configuration

After adding the bot to your group, you should configure the following settings:

  1. Set a welcome message using /welcome command
  2. Set group rules using /rules command
  3. Configure moderation settings using /settings command

Permission Levels

The bot uses a hierarchical permission system to control who can use certain commands:

Level Value Description Permissions
Owner 1000 Group creator or designated owner Full access to all commands
Admin 100 Group administrators Can use all moderation commands
Moderator 50 Assigned moderators Can mute, warn, and configure some settings
Trusted 10 Trusted members Immune to some automatic moderation
Regular 0 Standard members Basic interaction commands
Restricted -10 Limited permissions Read-only access
Banned -100 Banned from the group No access

Moderation

The bot provides several tools for moderating your group:

Warning System

Users can be warned for inappropriate behavior using the /warn command. After a certain number of warnings (configurable), actions like mute or ban can be automatically applied.

Automated Moderation

When enabled, the bot can automatically detect and address:

  • Spam and excessive message frequency
  • Inappropriate language
  • Unauthorized links or forwarded messages

Manual Actions

Moderators and admins can manually:

  • Ban users from the group (/ban)
  • Temporarily mute users (/mute)
  • Remove restrictions (/unmute)
  • Allow banned users back (/unban)

Settings

The bot has several configurable settings:

Group Settings

  • moderation_enabled - Enable/disable automatic moderation (on/off)
  • auto_delete - Automatically delete command messages after execution (on/off)
  • welcome_enabled - Enable/disable welcome messages (on/off)
  • max_warnings - Number of warnings before automatic action (1-10)

To change a setting, use the command:

/settings [setting_name] [value]

Troubleshooting

Common Issues

  • The bot doesn't respond to commands

    Solution: Make sure the bot is properly added to the group and has the necessary permissions.

  • Unable to use moderation commands

    Solution: Check that you have the required permission level to use those commands.

  • Welcome messages aren't working

    Solution: Verify that welcome_enabled is set to "on" in settings.

Getting Help

If you encounter issues not covered in this documentation, you can:

  • Use the /help command in Telegram for a quick reference
  • Check for updates on the bot's channel
  • Contact support if the issue persists
