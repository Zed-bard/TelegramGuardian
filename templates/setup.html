Configure the Telegram Group Manager Bot

Bot Configuration

Follow these steps to set up and configure your own instance of the Telegram Group Manager Bot.

1
Create a new bot with BotFather

First, you need to create a new bot on Telegram by talking to the BotFather:

  1. Open Telegram and search for @BotFather
  2. Send the command /newbot
  3. Follow the instructions to choose a name and username for your bot
  4. BotFather will give you a token - copy this token
2
Enter your bot token

Your token is used to authenticate your bot with the Telegram API. Keep it secret and never share it publicly.
3
Add bot to your group

After configuring your bot token, you can add it to your Telegram group:

  1. Open your Telegram group
  2. Click on the group name at the top to access group info
  3. Select "Add members"
  4. Search for your bot by username
  5. After adding, make the bot an admin with appropriate permissions
4
Start using bot commands

You can now start using commands in your group:

  • /start - Initialize the bot
  • /help - Show available commands
  • /welcome - Set a welcome message for new members
  • /rules - Set group rules
  • /settings - Configure bot settings

Optional Settings

Bot Permissions

For your bot to function properly in groups, it needs certain permissions. When adding the bot to a group, make sure to grant it the following permissions:

  • Delete messages
  • Ban users
  • Invite users via link
  • Pin messages
  • Manage voice chats

Advanced Configuration

You can customize the bot's behavior by modifying the configuration file directly:

{ "moderation_enabled": true, "auto_delete": false, "welcome_enabled": true, "welcome_message": "Welcome {username} to {group_name}! Please read the group rules." }
